# Smart Mirror – Glow Guide

## Overview

Glow Guide is an AI-powered Smart Mirror application that analyzes facial skin conditions using Computer Vision techniques. The system helps identify skin concerns and provides basic skincare recommendations.

## Features

- Face Detection
- Skin Analysis
- AI-Based Recommendations
- Computer Vision
- Real-Time Processing

## Technologies

- Python
- OpenCV
- NumPy
- Google Colab

## Future Improvements

- Deep Learning Skin Classification
- Mobile Application
- Real-Time Webcam Support
- Personalized Skincare Suggestions
